# 🎨 Arte Front — Angular CRUD App

Aplicação Angular 17 para gerenciamento de Artes, com operações completas de CRUD consumindo uma Rest API.

🔗 **Acesse a aplicação online:** [https://arte-front.vercel.app/login](https://arte-front.vercel.app/login)

---

## 🚀 Como rodar localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/rodrigo749/arte-front.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd arte-front
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   ng serve --port 8081
   ```

5. **Acesse no navegador:**
   ```
   http://localhost:8081/
   ```

## 🛠️ Tecnologias utilizadas

- [Angular 17](https://angular.io/) — Framework principal
- [TypeScript](https://www.typescriptlang.org/) — Linguagem de programação
- [Bootstrap](https://getbootstrap.com/) — Estilização e responsividade
- [RxJS](https://rxjs.dev/) — Programação reativa
- [HttpClient](https://angular.io/guide/http) — Consumo de APIs REST
- [Vercel](https://vercel.com/) — Deploy e hospedagem

## 📋 Funcionalidades

- Cada **Arte** possui: `id`, `título`, `descrição` e `status de publicação`
- ✅ Criar uma Arte
- 📄 Listar todas as Artes
- ✏️ Editar uma Arte
- 🗑️ Deletar uma Arte
- 🔍 Buscar Artes pelo título
